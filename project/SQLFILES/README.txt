to connect to mysql in XAMPP from the terminal use this:

/Applications/xampp/xamppfiles/bin/mysql -u root -p

to setup the database for cpus, open the cpu.txt file, then copy and paste the
two sql commands into the terminal connection to sql above.

or navigate to the pcparts.sql file and run this command:

/Applications/xampp/xamppfiles/bin/mysql -u root -p < pcparts.sql
